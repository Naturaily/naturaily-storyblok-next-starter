## ðŸ¤“ Usage

1. Import this package in the other package or app by adding it as a dev dependency to `package.json` file:

```json
"devDependencies": {
    "tailwind-config": "*",
}
```

2. Import the default settings to the `tailwind.config.js` file of particular package/app:

```js {1, 5}
const defaultConfig = require('tailwind-config/tailwind.config');

/** @type {import("tailwindcss").Config} */
const config = {
  presets: [defaultConfig],
};

module.exports = config;
```

3. Import the default settings to the `postcss.config.js` file of particular package/app:

```js
module.exports = require('tailwind-config/postcss.config');
```

4. Import the `global.css` file into the global place file in your application (ex. `layout.tsx`):

```tsx
import 'tailwind-config/global.css';
```

5. Use [tailwind classes â†—](https://tailwindcss.com/docs/aspect-ratio) in your components within the package/app you implemented the config to:

```tsx /w-full my-5 p-5 md:my-10/
<Component className="w-full my-5 p-5 md:my-10" />
```

## ðŸ¤“ Extending the default settings

You can always extend and/or overwrite the default settings in the `tailwind.config.js` file of particular package/app, for example:

```js {7-13}
const defaultConfig = require('tailwind-config/tailwind.config');

/** @type {import("tailwindcss").Config} */
const config = {
  presets: [defaultConfig],
  theme: {
    extend: {
      colors: {
        primary: {
          500: #000000
        }
      }
    }
  }
};

module.exports = config;
```
