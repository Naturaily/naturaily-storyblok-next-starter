# Docs Contribution Guide

Welcome to the Naturaily storyblok starter Docs Contribution Guide! We're excited to have you here.

Our goal is to ensure that everyone in the community feels empowered to contribute and improve our docs and application.

## How to Contribute

The docs content can be found on the our [repository](https://github.com/Naturaily/naturaily-starter). To contribute, you can edit the files directly on GitHub or clone the repo and edit the files locally.

<Steps>
### Fork this repository

### Create your branch

```bash copy
git checkout -b my-awesome-contribution
```

### Commit your changes

```bash copy
git commit -m 'feat: Add some awesome contribution'
```

### Push to the branch

```bash copy
git push origin my-awesome-contribution
```

### Open a pull request in our repository

</Steps>

## Extensions for VSCode

We also recommend the following extensions for VSCode users:

- [MDX](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx): Intellisense and syntax highlighting for MDX.
- [Grammarly](https://marketplace.visualstudio.com/items?itemName=znck.grammarly): Grammar and spell checker.
- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode): Format MDX files on save.
- [Eslint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint): ESLint statically analyzes your code to quickly find problems.
- [GraphQL: Language Feature Support](https://marketplace.visualstudio.com/items?itemName=GraphQL.vscode-graphql): GraphQL extension for VSCode built with the aim to tightly integrate the GraphQL Ecosystem
